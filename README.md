# Connect4
This is a text based version of Connect 4.

## Design

To view the docs locally you can run mkdocs 
```
brew install python
python3 -m venv venv
source venv/bin/activate
python3 -m pip install mkdocs mkdocs-mermaid2-plugin
mkdocs serve
```
